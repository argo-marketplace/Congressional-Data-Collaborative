result.txt - the main results of the census api script
Variables.csv - A list variable codes and their desriptions from the 2017 ACS 5-year  tables.
ref_legislators.csv - A list of congress members and their information sourced from https://github.com/unitedstates/congress-legislators
state_codes.csv - A reference table of State Codes and State Names

census_api_cds_script.sh - A script that queries the US Census API
vars.txt - A file used by this script that contains a list of census variables. Change this to change what to query for.
cds.txt - A file used by this script to list the state codes and congressional district codes for each state.

state_cd_relationship.xlsx - A file that was used to determine cds.txt
